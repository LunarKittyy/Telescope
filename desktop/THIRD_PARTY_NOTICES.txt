Python runtime dependencies
-----------------------------

These are installed from PyPI at setup time (desktop/requirements.txt) - on
Linux by start.sh into a Telescope-owned virtual environment, on Windows
they are compiled into TelescopeDesktop.exe by PyInstaller. Each project's
own repository/PyPI page has the authoritative license text.
